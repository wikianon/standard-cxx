Usage

GuruVision [-h host-log-file{,host-log-file}]

Introduction

GuruVision is the log file viewer for the VirtualGuru application as part of The XE Toolkit. VirtualGuru generates text files that are readable by a simple text viewer, however, more structured information is presented to the user when viewed using the GuruVision application.

When started without arguments, the user is presented with a file chooser dialog box to select the name of a log file to view.

Features

Tabbed Display

The top level display of the GuruVision GUI contains a tabbed pane with each host-log-file viewed in a separate tab pane identified by the name of the host-log-file on the tab at the top of the pane. A bordered button collection allows the user to select which records are to be viewed, either All or only those records identified by one of the seven state colors. Within the table containing the records, a Details button will display the complete record in the text pane at the bottom of the pane. Navigation buttons below the records table allow movement through the host-log-file and the position within the log file is displayed above the navigation buttons.

GuruVision does not load the entire host-log-file into memory for navigation. Only 10 records at a time are loaded and therefore movement through the file may be delayed when the host-log-file is located on a network disk.

Log File Location

The log files for GuruVision are stored in the directory named VirtualGuru under the logs directory at the top level of the XE installation. The name of the log file will be the name of the monitored host.
