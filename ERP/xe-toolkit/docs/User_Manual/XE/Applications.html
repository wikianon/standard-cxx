Applications

The XE Toolkit contains both command-line and GUI applications. They exist as both client applications and administrative applications for starting and querying the status of CaptiveCollector. The following table lists the directories that hold the different types of applications on the supported platforms. The directory names listed are below the top level of the installation directory of The XE Toolkit.

Application List

CacheStat

Command line, an interval/iterations based program that displays the effectiveness of caches on the monitored host.

Counters [remote-host]

GUI, a dialog much like the counter selection dialog for the performance registry on Windows, only this one communicates with CaptiveCollector.

CpuGraph [-h remote-host]

GUI, a simple graph application that displays CPU use of the monitored host and allows the graph type to be changed between area, line and bar.

CpuMeter [-h remote-host]

GUI, an XE version of the SE CpuMeter which displays CPU use of the monitored host.

CpuPie [-h remote-host]

GUI, a pie chart version of the CpuGraph.

Disks [-h remote-host]

Command line, display the disks known by CaptiveCollector on the monitored host.

Explain { [-l(ist) host] | host category field }

Command line, list either all of the categories and counter names of the specified host or list the explanation field for the specified counter. At no time should this command be run against a CaptiveCollector running on Windows. The collector will grow to over 200 megabytes resident.

GuruVision

GUI, GuruVision is the log file viewer for the VirtualGuru application.

IoStat

Command line, an interval/iterations based program that displays the activity of logical and physical disks on the system.

MemoryPie [-h remote-host]

GUI, a pie chart showing the utilization of physical memory.

MpStat

Command line, an interval/iterations based program that displays the activity of all CPUs on a multi-processor system.

NicStat

Command line, an interval/iterations based program that displays the activity of the network interfaces on a system.

OSFlavor [-h remote-host]

Command line, show the operating system name of the specified host.

PageFilePie [-h remote-host]

GUI, a pie chart showing the utilization of the page file space.

SysInfo [-h remote-host]

Command line, display a summary of the hardware and operating system configuration of the specified host.

SysStat

Command line, an interval/iterations based program that displays information about varied system activity.

TcpStat

Command line, an interval/iterations based program that displays information about activity in the TCP stack.

Top

GUI, Top provides a network aware interface for monitoring the processes on multiple computers at once.

VirtualGuru

Command line, a background process that applies performance rules to a monitored host and writes the results to a log file.
